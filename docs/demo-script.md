# Demo Script

1. `make dev`
2. Open:
   - http://localhost:3000 (Ops panel)
   - http://localhost:8080/health/ready
   - http://localhost:9090 (Prometheus)
   - http://localhost:3001 (Grafana)
3. Generate activity: `./scripts/demo-traffic.sh`
4. Present pages:
   - Service Overview
   - Live Metrics
   - Error Explorer
   - Trace Explorer
   - Config & Environment
